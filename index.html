Modern Windows protection toolkit

Control HOSTS, DNS and firewall protection from one desktop app.

HostShield Manager is a WinUI 3 application for managing the Windows HOSTS file, domain block lists, whitelist rules, firewall blocking, Secure DNS / DoH, diagnostics, profiles, logs and live network activity.

Main features

Everything needed for local domain and network control.

The interface brings common Windows protection workflows into one organized workspace instead of forcing users to edit system files and open separate tools manually.

Hosts Editor

View, edit, validate and repair HOSTS lists, manage categories, comments, duplicates, whitelist entries and reusable profiles.

Domain Diagnostic

Check DNS resolution, HTTP, HTTPS port access, SSL certificate state, HOSTS status, whitelist state, ping and optional Whois.

Firewall Blocker

Scan folders, select detected executables and create or remove inbound and outbound Windows Firewall rules.

Secure DNS / DoH

Inspect DNS encryption, view adapter details, apply recommended providers, set manual DNS servers or reset to DHCP.

Netstat Monitor

View TCP, TCP6, UDP and UDP6 connections with process, PID, address, state, executable path and GeoIP country data when available.

Profiles, Logs & Settings

Save and restore HOSTS states, review activity logs, use backups and control safety prompts, snapshots and startup self-checks.

Workflow design

Built for actions that touch sensitive Windows settings.

HostShield Manager is designed around controlled changes, clear status information and local recovery data. Administrator privileges are required when Windows HOSTS, DNS or Firewall settings need to be changed.

  • Backups before many HOSTS operations.
  • Local logs for startup, DNS, firewall, profile and HOSTS events.
  • Whitelist warnings to prevent accidental blocking of trusted domains.
  • Portable layout with user settings, profiles, backups and translation files.

Quick Actions

Fast tools for blocking one or more domains, adding manual HOSTS entries, whitelisting domains and running diagnostics without switching through every module.

Profiles

Save the active Windows HOSTS file, import or export profiles, load a profile, open it in Notepad and restore built-in default or recommended baselines.

Logging

Search, filter, copy, delete or clear application events and open the current log file when troubleshooting write failures, DNS commands or firewall operations.

Localization

Language files are loaded from the portable Language folder, with English and Romanian included and repair attempts when required files are missing or invalid.
